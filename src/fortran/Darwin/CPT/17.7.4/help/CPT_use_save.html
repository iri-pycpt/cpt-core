Saving the Results

Most of the results can be saved in files by using the Output Results option under the File menu item. This option becomes accessible only after the calculations are complete. A variety of results can be saved:

  • Input Data
  • Climatologies (for the Y data)
  • Modes (if the CCA or PCR options are used)
  • Hindcasts
  • Forecasts
Skill maps and verification results can be saved also, but only by right-clicking on the skill map window, or a verification window. They cannot be saved from the standard output files window because the skill scores and the verification results are stored in temporary memory. Forecasts (including principal component scores for new values of the predictors that are not part of the training period) can be saved only once a forecast has been viewed using one of the Tools ~ Forecast menu items.

File names for each of the desired outputs need to be specified. The default output format is tab-separated text files in the latest CPT version format, but binary files, GrADS formats and other formatting options are available from the Edit ~ File formats menu. For the default CPT output, it is possible to suppress the output of the CPT tags using Options ~ Include CPT tags, which has a tick mark if the output of CPT tags is activated. Further details about the output file formats is available from Edit ~ File formats.
