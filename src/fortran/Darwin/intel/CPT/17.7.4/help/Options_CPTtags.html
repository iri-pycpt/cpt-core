133: Include CPT tags

If data are saved in the default formatted sequential output format, then the data will be saved in CPT version 16 format. Depending upon the type of data in the file, it should be possible to read this output file back into CPT, as long as the CPT tags are output. If you want to use the output file for some other purpose, you may wish to exclude the CPT tags, in which case you can suppress the output of the tags by selecting Options ~ Include CPT tags. This menu item toggles the output of the tags on and off: if the tags are to be output, tick appears beside the menu item, but if the output is to be suppressed there is no tick.
