221: Start at

To change the date of the first data in the X file to be used, type the desired dates in the respective boxes, or using the attached fly-wheels. CPT may automatically adjust the length of the training period, if appropriate, but will not attempt to reset the start-at date for the predictand (Y) file or for the new predictors (Z) file.
