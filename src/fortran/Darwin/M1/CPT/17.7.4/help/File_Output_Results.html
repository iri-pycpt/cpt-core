111: Output Results

Most of the results can be saved in files by using the Output Results option under the File menu item. This option becomes accessible only after the calculations are complete. A variety of results can be saved, depending upon which View option is active (the file number is for batch-version users and is not indicated in the Windows version):

  • Input Data
    • (101) X input data with missing values replaced
    • (111) interpolated X data
    • (102) Y input data with missing values replaced
    • (112) transformed Y data
    • (121) Y input data converted to category identifiers (1 = below-normal; 2 = normal; 3 = above-normal)
    • (122) Y input data converted to category identifiers retroactively (1 = below-normal; 2 = normal; 3 = above-normal)
    • (122) target period Y input data converted to category identifiers (for PFV) (1 = below-normal; 2 = normal; 3 = above-normal)
  • Hindcasts
    • (201) cross-validated predictions
    • (205) Fitted Y values (the transformed fitted values are output if the Transform Y Data option is on)
    • (211) retroactive deterministic predictions
    • (212) retroactive ensemble predictions
    • (213) retroactive prediction limits (2 = upper and 1 = lower limits of the prediction intervals)
    • (214) retroactive probabilistic predictions
    • (215) retroactive forecast odds
    • (216) retroactive prediction error variances
    • (221) double cross-validated deterministic predictions
    • (222) double cross-validated ensemble predictions
    • (223) double cross-validated prediction limits (2 = upper and 1 = lower limits of the prediction intervals)
    • (224) double cross-validated probabilistic predictions
    • (225) double cross-validated forecast odds
    • (226) double cross-validated prediction error variances
  • Modes (if the CCA or PCR options are used)
    • (301) X eigenvalues
    • (302) X spatial loadings
    • (303) X temporal scores
    • (311) Y eigenvalues
    • (312) Y spatial loadings
    • (313) Y temporal scores
  • CCA Results
    • (401) canonical correlations
    • (411) X CCA map loadings
    • (412) X CCA map series
    • (421) Y CCA map loadings
    • (422) Y CCA map series
  • PCR Results
    • (432) PC regression coefficients (coefficients between Y and the X EOFs)
    • (431) regression coefficients (expanded coefficients between Y and X)
  • MLR Results
    • (431) regression coefficients
  • GCM Results
    • (431) regression coefficients
  • Forecasts
    • (501) forecast probabilities
    • (502) forecast odds
    • (511) deterministic forecasts
    • (5111) observed (i.e., persisted) component of the forecasts (if the persistance option is on)
    • (5112) predicted component of the forecasts (if the persistance option is on)
    • (512) ensemble forecasts
    • (513) prediction limits (1 = lower and 2 = upper limits of the prediction intervals)
    • (514) prediction error variances
    • (521) probabilities of exceedance or non-exceedance
    • (531) predictor time scores (values of the updated X EOFs)
    • (532) predictor CCA time scores (values of the updated X CCA modes)
    • (541) interpolated new X data
  • Climatologies (for the Y data)
    • (601) thresholds (1 = lower tercile, or boundary between below-normal and normal; 2 = upper tercile, or boundary between normal and above-normal)
    • (602) averages
    • (604) standard deviations
    • (605) coefficients of variation (if the zero-bound is on).
    • (603) correlations between each of the predictands and each predictor field (sometimes known as teleconnection maps)
    • (611) X data (standardized depending on the X Mode settings)
Skill maps and verification results can be saved also, but only by right-clicking on the skill map window, or a verification window. They cannot be saved from the standard output files window because the skill scores and the verification results are stored in temporary memory. Forecasts (including principal component scores for new values of the predictors that are not part of the training period) can be saved only once a forecast has been viewed using one of the Tools ~ Forecast menu items. The following results are available:

  • Scores
    • (701) goodness index
    • (711) skill scoes
    • (712) p-values
    • (713) ROC results
    • (721) probabilistic scores
    • (723) probabilistic ROC results
    • (724) reliabiity results
    • (725) ranked-hits scores
    • (726) weather roulette results
File names for each of the desired outputs need to be specified. The default output format is tab-separated text files in the latest CPT version format, but binary files, GrADS formats and other formatting options are available from the Edit ~ File formats menu.
