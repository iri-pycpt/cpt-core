Axis Limits

There is some flexibility to adjust the Y-axis limits used on graphs. The Customise ~ Axis Limits or Customise ~ Y Axis Limits menu item can be activated by right-clicking on the graphic. Select the relevant graphic (if prompted) and set new maximum and minimum values for the axis. The number and intervals of tick marks will be reset automatically; options to set the number of major and minor tick marks are planned for a future version of CPT. For Scatter Plots, resetting the axis limits will reset both axes.
